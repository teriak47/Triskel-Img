# Triskel-Img

Dépôt d'images pour les projets Triskel / Image repository for Triskel projects

## Description

Ce dépôt sert uniquement à héberger des images qui peuvent être référencées dans d'autres projets ou documentation.

This repository is used solely to host images that can be referenced in other projects or documentation.

## Structure

```
images/
├── screenshots/    # Captures d'écran / Screenshots
├── logos/          # Logos
├── icons/          # Icônes / Icons
├── banners/        # Bannières / Banners
└── misc/           # Autres images / Miscellaneous images
```

## Utilisation / Usage

Pour référencer une image dans vos projets, utilisez l'URL brute GitHub :

To reference an image in your projects, use the raw GitHub URL:

```
https://raw.githubusercontent.com/teriak47/Triskel-Img/main/images/<category>/<filename>
```

### Exemple / Example

```markdown
![Alt text](https://raw.githubusercontent.com/teriak47/Triskel-Img/main/images/logos/my-logo.png)
```

## Contribution

Pour ajouter une image :

1. Choisissez la catégorie appropriée dans le dossier `images/`
2. Uploadez votre image avec un nom descriptif
3. Créez une pull request

To add an image:

1. Choose the appropriate category in the `images/` folder
2. Upload your image with a descriptive name
3. Create a pull request

## Formats supportés / Supported Formats

- PNG
- JPG/JPEG
- GIF
- SVG
- WebP
